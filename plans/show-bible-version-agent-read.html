Show Bible Version When Displaying a Verse

Goal

When a Bible verse is displayed, also show its translation/version next to the reference, e.g. John 3:16 (KJV).

Research

  • BibleControlPanel.jsx already passes bible: currentBible?.name into onSelectVerse.
  • LyricDisplayApp.jsx handleBibleVerseSelect ignores verseData.bible.
  • StageOutput.jsx and RegularOutput.jsx render bibleReferenceText but never the version.
  • LyricsStore has no version field.

Files to Modify

  1. src/context/LyricsStore.js — add bibleVersion state + setBibleVersion; clear it inside setLyricsFileName; persist it.
  2. src/components/LyricDisplayApp.jsx — call setBibleVersion(verseData.bible).
  3. src/pages/StageOutput.jsx — render reference (version).
  4. src/pages/RegularOutput.jsx — same.

Display Format

reference (version), only when a version exists.

Tests

  • Add pure helper formatBibleReference(reference, version) and unit-test it.
  • Run lint, typecheck, and the test suite.

Phases

  1. Store field + setter + reset-on-filename.
  2. Wire into handleBibleVerseSelect.
  3. Helper + unit test.
  4. Render in both outputs.
  5. Verify.
